# ICT Model 12: Order Block + Fair Value Gap (OB+FVG) Scalping Model

## Overview
ICT Price Action Model Number 12 is a precision scalping model designed to capture 20-pip movements through the strategic combination of Order Blocks and Fair Value Gaps. This model emphasizes "bread and butter" setups rather than attempting to capture absolute highs or lows.

## Core Objective
- **Target**: 20 pips per trade
- **Philosophy**: Consistent, reliable profits rather than home runs
- **Approach**: Wait for high-probability setups rather than forcing trades

## Key Components

### 1. Order Block Identification
- **Bearish Order Block**: Last two up-close candles before a significant move down
- **Bullish Order Block**: Last two down-close candles before a significant move up
- Must show clear institutional involvement (large volume, clean breaks)

### 2. Fair Value Gap (FVG) Requirements
- **Critical Rule**: The Fair Value Gap must NOT be directly linked to the Order Block candle
- **Separation Principle**: There must be clear separation between the OB and FVG formation
- **Quality Filter**: FVG should form during expansion moves, not consolidation

## Trading Sequence Protocol

### Phase 1: Order Block Formation
1. Identify last two opposing candles (up-close for bearish, down-close for bullish)
2. Wait for price to move away from the order block
3. Confirm institutional involvement through volume and clean breaks

### Phase 2: Retest and Expansion
1. Price returns to test the order block
2. **Expansion Price Swing**: Price accelerates away from the order block
3. **DO NOT CHASE**: Wait for the retracement setup

### Phase 3: Fair Value Gap Formation
1. Identify FVG during the expansion move
2. Ensure FVG is separate from the original order block candle
3. Wait for price to retrace back to the FVG

### Phase 4: Entry Execution
- **Short Entry**: At the high of the Fair Value Gap (for bearish setups)
- **Long Entry**: At the low of the Fair Value Gap (for bullish setups)
- **Target**: 20 pips from entry point
- **Stop Loss**: Above/below the Fair Value Gap with gap closure allowance

## Risk Management Framework

### Stop Loss Placement
- **Default**: Above the Fair Value Gap high (shorts) or below FVG low (longs)
- **Gap Closure Allowance**: Factor in potential full gap closure
- **Mental Preparation**: Expect gap fills and trust the stop loss protocol
